# SwissArmyPM

A lightweight desktop project management tool with AI integration.

## 🚀 Getting Started

\`\`\`bash
npm install
npm run dev
\`\`\`

## 📁 Project Structure

\`\`\`
SwissArmyPM/
├── docs/              # Documentation
├── design/            # Design system
├── src/
│   ├── main/         # Electron main process
│   ├── renderer/     # React frontend
│   └── shared/      # Shared types
└── templates/        # App templates
\`\`\`
