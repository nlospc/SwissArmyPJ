## 🎯 项目特点

- 🚀 **轻量级** - 基于 Electron 的桌面应用，资源占用低
- 🎨 **现代化 UI** - 使用 Ant Design v5 组件库
- 🧠 **AI 集成** - 内置 AI 辅助功能
- 📊 **本地数据** - SQLite + FTS5 全文搜索
- 🔧 **易扩展** - 模块化架构设计
